🧪 Лабораторная работа №8

Тема: Интеграция асинхронного сервиса в веб-приложение

⸻

🎯 Цель работы

Реализация и подключение асинхронного сервиса для обработки запросов веб-приложения.

⸻

🧩 Содержание работы

В рамках лабораторной работы был разработан отдельный асинхронный сервис на языке Python, предоставляющий один HTTP-эндпойнт. Сервис предназначен для выполнения вычислений/обработки данных вне основного бэкенда и взаимодействует с ним по HTTP.

Основной бэкенд отправляет запрос в асинхронный сервис и получает результат обработки в формате JSON.

⸻

🧠 Краткое описание реализации
	•	Асинхронный сервис реализован с использованием FastAPI.
	•	Используется асинхронная модель обработки запросов (async/await).
	•	Реализован один эндпойнт POST, принимающий входные данные и возвращающий результат обработки.
	•	Обмен данными осуществляется по HTTP в формате JSON.
	•	Сервис может быть развернут отдельно от основного приложения.

⸻

📌 Итог

В результате выполнения лабораторной работы был реализован и подключён асинхронный сервис на Python, обеспечивающий вынесение части логики обработки данных из основного бэкенда и демонстрирующий принципы сервисного и асинхронного взаимодействия.
