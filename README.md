## PANKREATITMED backend

Данный репозиторий содержит серверную часть веб-приложения, реализующую бизнес-логику системы, REST API и взаимодействие с внешними сервисами.

---

## 📌 Назначение проекта
Бэкенд предназначен для обслуживания SPA-приложения и обеспечивает:
- работу с сущностями предметной области,
- обработку пользовательских заявок,
- авторизацию и разграничение прав доступа,
- взаимодействие с асинхронным сервисом обработки данных.

---

## ⚙️ Реализованный функционал

- REST API для работы с основными сущностями системы  
- Регистрация и аутентификация пользователей  
- Авторизация на основе ролей (гость, пользователь, модератор)  
- Ограничение доступа к методам API по правам  
- Работа с заявками (создание, редактирование, просмотр, завершение)  
- Автоматическое связывание заявок с авторизованным пользователем  
- Интеграция с асинхронным сервисом (HTTP-взаимодействие)  
- Документирование API с помощью Swagger  

---

## 🛠 Используемые технологии

- Go
